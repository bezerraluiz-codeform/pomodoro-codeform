# Novas Features

- Autenticação Login
- Tasks salvas em banco
- Tasks compartilhadas
- Criação de workspace em grupo
- Usuários e grupo pré-definidos
- Tasks em coluna (coluna por usuário, exemplo, coluna 1 com nome do usuário 1) com card de responsável, status, nome da taks e time SLA

# Tech Stack

- Supabase
- Next
- TypeScript
- TailwindCSS

## Usuário

1. Entra no pomodoro e efetua o login com sua conta pré-cadastrada
2. Consegue visualizar os usuários do workspace, tasks de cada — atualizadas em real time, e o timer rodando em real time
3. Consegue atualizar somente as taks que estão como responsável ele mesmo
4. O timer só pode ser pausado pelos usuários administradores do workspace
